# BIT Research Project

The recommender uses [AH Allerhande](https://www.ah.nl/allerhande) data for recipes.
All generated data is applicable to resident of the Netherlands.

## Flow

* Scrape (with `olive`)
* Process (with `olive`)
* Generate (with `olive`)
* Recommend (with `poireau`)
